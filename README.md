<!-- Installation -->

Install my-project with npm

```bash

```


<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/kevinh-e/tastebuds.git
```

Go to the project directory

```bash
  cd tastebuds
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
